# Задание 7 на чистом Python

Это решение выполнено без Flask: веб-интерфейс реализован на стандартной библиотеке Python через `http.server`, а работа с PostgreSQL идет через `psycopg2`.

## Что реализовано
- Страница входа.
- Главная страница.
- Просмотр таблиц БД.
- Добавление записей.
- Обновление записей.
- Логирование неудачных попыток входа.
- Dockerfile.
- Docker Compose с PostgreSQL.
- Параметризованные SQL-запросы.

## Запуск
```bash
docker compose up --build
```

После запуска откройте:
```text
http://localhost:5000/login
```

## Данные для входа
- Логин: `app_user`
- Пароль: `secure_password_123`

## Проверка
1. Войдите в приложение.
2. Откройте просмотр таблиц.
3. Добавьте запись.
4. Обновите существующую запись.
5. Попробуйте войти с неверным паролем и затем проверьте файл `logs/auth.log`.

## Полезные команды
```bash
docker compose down
docker compose down -v
docker compose logs -f app
cat logs/auth.log
```
